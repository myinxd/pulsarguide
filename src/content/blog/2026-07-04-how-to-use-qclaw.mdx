## QClaw 是什么？

QClaw 是腾讯电脑管家团队基于开源 [OpenClaw](https://github.com/openclaw/openclaw) 框架打造的本地 AI Agent 助手，中文昵称"**小龙虾**"。它的核心能力用一句话说就是：

**在你的电脑本地跑一个 AI，它听得懂你的自然语言指令，能操控电脑执行真实任务——而且你可以在微信里远程指挥它。**

和普通的聊天 AI 不同，QClaw 不是"给你一段文字回复"，而是**真的在帮你干活**：整理文件、发邮件、写代码、操控浏览器……

## 方式一：电脑上用（Mac / Windows）

### 第一步：下载安装（1 分钟）

打开官网 **https://qclaw.qq.com/**，根据你的系统选择对应版本：

| 你的电脑 | 下载哪个 |
|---------|---------|
| Mac（M1/M2/M3/M4 芯片） | macOS (Apple 芯片) |
| Mac（Intel 芯片） | macOS (Intel 芯片) |
| Windows 10 / 11 | Windows |

下载后：
- **Mac**：双击 dmg → 拖图标到「应用程序」文件夹
- **Windows**：双击 exe → 一路点「下一步」

### 第二步：首次启动配置（1 分钟）

首次打开 QClaw，会经历几步引导：

1. **扫码绑定微信**（核心步骤）  
   打开微信 → 扫描屏幕上二维码 → 确认授权

2. **选择大模型**  
   默认使用腾讯混元大模型，每天有 **4000 万免费 Token**，普通用法完全够用。如果你想用自己申请的 API Key（DeepSeek、千问、豆包等），在设置里填入即可。

3. **安装 Skills（可选）**  
   Skills 就是 QClaw 的技能包，比如"自动写周报"、"帮我美化输出格式"。新手可以先跳过，之后随时在"灵感广场"一键安装。

### 第三步：发出第一条指令

绑定完成后，你会看到两个界面：

- **对话界面**：直接打字和 QClaw 聊天
- **工作室**：能看到 QClaw 正在执行的子任务和进度

常见的上手指令示例：

```
帮我整理一下桌面上的文件，按类型分类
分析一下我电脑安装了哪些软件
把下载文件夹里一周前的文件找出来
```

## 方式二：微信小程序里用（手机远程操控）

这是 QClaw 最有意思的地方——**不用坐在电脑前，手机发消息就能让电脑干活**。

### 安装小程序

打开微信 → 搜索「**QClaw**」或「**腾讯小龙虾**」→ 找到小程序并打开。

> ⚠️ 如果搜索不到，说明你的 QClaw 客户端还没更新到最新版（v0.1.9+ 才支持小程序）。回到电脑端检查更新，重启一次客户端即可。

### 连接电脑

1. 打开电脑端 QClaw，确保在同一局域网或已登录同一微信账号
2. 小程序里点「连接设备」→ 选择你的电脑
3. 连接成功后，你会看到一个类似客服消息的界面

### 能做什么？

- 在地铁上让 QClaw 帮你打开某个文件
- 出门前让它把今天的日报草稿发到你邮箱
- 周末躺床上想起有个报表没做，手机发条消息，回去发现已经搞定了

> **隐私提示**：QClaw 本地运行，数据不会上传到腾讯服务器。但给它的权限越大，它能做的事情越多——建议按需授权，不用一开始就全开。

## 常见坑点排错

| 问题 | 解决方法 |
|------|---------|
| 小程序搜不到 QClaw | 更新电脑端到最新版本，重启 |
| 微信扫码一直失败 | 检查微信是否已登录，尝试重新扫码 |
| Token 用完了 | 在设置里填入自己的 API Key |
| 工作室加载不出来 | 重启 QClaw 客户端 |
| Mac 提示"无法打开" | 在「系统设置 → 隐私与安全性」里手动允许 |

## 我的真实使用感受

用了一周之后，我发现 QClaw 最实用的场景就三个：

1. **重复性文件整理**：每周要整理一次周报数据，让它自动分类归档，比我自己拖拽快多了
2. **出门在外的临时需求**：在地铁上想起有个链接要发给同事，手机上让 QClaw 帮我发
3. **懒人周报**：设一个定时任务，每周五自动把每天的日志汇总成周报草稿

技术上它基于 OpenClaw，底层是本地运行的 AI Agent，所以**数据不经过云端**，隐私上比很多"AI助手"更安全。对于我这种既想要 AI 能力又不想把工作数据交给第三方的人来说，刚刚好。
